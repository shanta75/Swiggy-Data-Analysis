# Swiggy-Data-Analysis
Swiggy Sales Analysis Dashboard built in Microsoft Excel using Pivot Tables, Pivot Charts, Slicers, KPI Cards, and Map Charts to analyze sales performance and customer trends.
## 📊 Project Overview
This project is an interactive Excel dashboard created to analyze Swiggy sales data and generate meaningful business insights. The dashboard provides a comprehensive view of sales performance, customer ratings, food categories, top-selling dishes, and state-wise sales distribution.
## 🚀 Features
* Total Sales Analysis
* Average Rating Analysis
* Order Count and Order Value Tracking
* Monthly Sales Trend Analysis
* Daily Sales Performance
* Food Category Analysis (Veg & Non-Veg)
* Top 10 Best-Selling Dishes
* State-Wise Sales Analysis using India Map Chart
* Interactive Slicers and Filters
## 🛠️ Tools Used
* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Map Charts
* Conditional Formatting
## 📈 Key Insights
* Identified top-performing food categories.
* Analyzed monthly sales patterns.
* Compared state-wise sales performance.
* Evaluated customer ratings and order trends.
